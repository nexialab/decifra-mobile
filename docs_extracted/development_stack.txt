1. VISÃO GERAL DO PRODUTO
1.1 Propósito
O DECIFRA é um aplicativo mobile de avaliação de personalidade baseado no modelo Big Five (IPIP-NEO-120), desenvolvido para digitalizar e escalar o método proprietário da Ártio. O sistema conecta três perfis: admin (Ártio), treinadoras comportamentais certificadas e clientes finais.
1.2 Problema Resolvido
Hoje as treinadoras aplicam o diagnóstico Big Five manualmente via planilhas, recebendo apenas números brutos sem interpretação visual ou recomendação de protocolos de intervenção. O app transforma esse processo em uma experiência completa, automatizada e cientificamente validada.
1.3 Diferencial Competitivo
Não é apenas o diagnóstico. O valor real está nos 90 protocolos de trabalho — planos de ação personalizados que o app sugere automaticamente com base nas combinações específicas das 30 facetas de personalidade identificadas, priorizando segurança emocional primeiro (Neuroticismo), depois execução, limites, energia e visão.
1.4 Identidade Visual
Baseada no ícone fornecido: - Conceito: Olho estilizado com espiral interna = “visão”, “profundidade”, “decifrar”, “introspecção” - Paleta: Gradientes de terracota/laranja queimado para vinho escuro - Significado: Calor humano, profundidade, transformação

// Pseudo-código da lógica de recomendaçãofunction recomendarProtocolos(resultados: Resultado[]): ProtocoloRecomendado[] {  const pontuacoes: Map<string, number> = new Map();    // 1. Pontuar por faceta individual  for (const r of resultados.filter(r => r.tipo === 'faceta')) {    let pontos = 0;    if (r.classificacao === 'Muito Alto' || r.classificacao === 'Muito Baixo') {      pontos = 3;    } else if (r.classificacao === 'Alto' || r.classificacao === 'Baixo') {      pontos = 2;    }        // Bônus de segurança: N1, N5, N6    if (['N1', 'N5', 'N6'].includes(r.codigo) && pontos > 0) {      pontos += 1;    }        pontuacoes.set(r.codigo, pontos);  }    // 2. Verificar combinações críticas  const combinacoes = [    { fatores: ['N', 'C5'], alto: ['N'], baixo: ['C5'],       protocolos: ['N6-A', 'C5-B'] }, // N alto + C5 baixo    { fatores: ['N1', 'C6'], alto: ['N1', 'C6'],       protocolos: ['N1-A', 'C6-A'] }, // N1 alto + C6 alto    // ... outras combinações  ];    for (const combo of combinacoes) {    if (matchCombinacao(resultados, combo)) {      for (const pid of combo.protocolos) {        pontuacoes.set(pid, (pontuacoes.get(pid) || 0) + 2);      }    }  }    // 3. Ordenar por prioridade e categorizar  const ordenados = Array.from(pontuacoes.entries())    .sort((a, b) => b[1] - a[1]);    return {    cliente: ordenados.slice(0, 3).map(p => ({ id: p[0], tipo: 'principal' }))      .concat(ordenados[3] ? [{ id: ordenados[3][0], tipo: 'ajuste_fino' }] : []),    treinadora: ordenados.slice(0, 3).map(p => ({ id: p[0], tipo: 'principal' }))      .concat(ordenados.slice(3, 5).map(p => ({ id: p[0], tipo: 'complementar' })))      .concat(ordenados[5] ? [{ id: ordenados[5][0], tipo: 'ajuste_fino' }] : [])  };}
